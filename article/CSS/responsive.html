bootstrap最核心的东西就是栅格系统啦，而栅格系统最核心的东西就是响应式原理，而响应式原理最核心的东西就是媒体查询啦，所以本篇文章来简单的了解下媒体查询及其相关的一些知识吧。

# 一、响应式所具有的特点

  1. 网页的宽度自动调整
  2. 尽量少用绝对宽度
  3. 字体要用rem，em作为单位
    • rem是相对于html标签计算（推荐使用）
    • em是相对于父元素计算
  4. 布局要使用浮动(v3.)，弹性布局(v4.)

# 二、@规则

  1. @charset 定义编码
  2. @import 引入css文件
  3. @font-face 自定义字体
  4. @keyframes animation里的关键帧
  5. @media 媒体查询 根据一个或多个基于设备类型，基本特点和环境来应用样式

# 三、@media 媒体查询 (opens new window)

  1. 媒体类型
    1. all 匹配所有设备类型
    2. print 打印机设备
    3. screen 彩色的电脑屏幕
    4. speech 听觉设备（针对有视力障碍的人士，可以把页面的内容以语音的方式呈现的设备）
  2. 媒体特性
    1. width 宽度
      1. min-width 最小宽度，宽度只能比这个大
      2. max-width 最大宽度，宽度只能比这个小
    2. height 高度
      1. min-height 最小高度，高度只能比这个大
      2. max-height 最小高度，高度只能比这个大
    3. orientation 方向
      1. landscape 横屏（宽度大于高度）
      2. partrait 竖屏（高度大于宽度）
    4. aspect-ratio 宽高比
    5. -webkit-device-pixel-ratio 像素比（webkit内核的私有属性）
  3. 逻辑运算符
    1. and 合并多个媒体查询（并且的意思）
    2. , 匹配某个媒体查询（或的意思）
    3. not 对媒体查询结果取反
    4. only 防范老旧浏览器，因为一些老旧浏览器可能并不认识@media ...媒体查询 它们会直接解析媒体查询里面的CSS样式
