    # 离散对数难题

    DH 密钥交换算法的安全性依赖于离散对数求解的困难

    全称 Discrete Logarithm Problem，简称 DLP

    # 前置知识

    # 原根

    1. 定义

      若 n、an、an、a 为正整数，且 n、an、an、a 互质，令 ad≡1(modn)a^{d} ≡ 1 (mod n)ad≡1(modn)，可知，使该式成立的 ddd 的取值可能多个，又由欧拉定理可知 ddd 中的一个取值一定为 φ(n)\varphi (n)φ(n) ，只不过此时并不确定 φ(n)\varphi (n)φ(n) 是否是 ddd 中所有取值中最小的那个。重点来了，如果此时所有 ddd 的可能取值中 φ(n)\varphi (n)φ(n) 恰好是最小的，那么就称 aaa 为模 nnn 的原根

      还有一个问题：当 nnn 为何值时，才会存在原根呢？因为并不是所有的素数都是有原根的

      答：只有当 n=1、2、4、pe、2pen=1、2、4、p^{e}、2p^{e}n=1、2、4、pe、2pe 时才会存在原根，其中 ppp 为奇素数， eee 为正整数

      重新看下下面对原根的定义：

      一个素数 qqq 的原根，为其各次幂能够产生从 111 到 q−1q−1q−1 所有整数

      也就是说，如果 aaa 是素数 qqq 的一个原根，那么数值 amodq,a2modq,...,aq−1modqa mod q, a^{2} mod q, ..., a^{q-1} mod qamodq,a2modq,...,aq−1modq 是各不相同的整数，并且以某种排列方式组成了从 111 到 q−1q-1q−1 的所有整数。

    2. 应用

      题目：已知若 aaa 与 nnn 互质，并且 aaa 是 nnn 的原根，求满足该式 ax≡1(modn)a^{x} \equiv 1 ( mod n )ax≡1(modn) 的 xxx 的最小值 ？

      答案：根据欧拉定理，若 aaa 与 nnn 互质，则 aφ(n)≡1(modn)a^{\varphi (n)} \equiv 1 ( mod n )aφ(n)≡1(modn)，又因为 aaa 是 nnn 的原根，所以 xxx 的最小值一定为 φ(n)\varphi (n)φ(n)，即 x=φ(n)x = \varphi (n)x=φ(n)

    # 离散对数

    1. 定义

      对于一个整数 bbb 和素数 qqq 的一个原根 aaa，可以找到惟一的指数 iii，使得 b=aimodqb=a^{i} mod qb=aimodq，其中 0≤i≤(q−1)0≤i≤(q−1)0≤i≤(q−1) ，指数 iii 称为 bbb 的以 aaa 为基数的模 qqq 的离散对数，该值被记为 inda,q(b)ind_{a, q}(b)inda,q​(b)

      也就是说，已知 b、a、qb、a、qb、a、q，要想求其对应的离散对数 iii ，几乎是不可能的，尤其是在素数 qqq 值很大的情况下

    2. 应用

      公钥密码学

    注意

    原根和离散对数这两个概念即可以从数论的角度来理解，也可以从抽象代数之群论的角度来理解

    # 模运算

    1. 定义

      “模”是“Mod”的音译，意为求余 给定一个正整数 ppp，任意一个整数 nnn，一定存在等式 n=kp+rn=kp+rn=kp+r 其中 k、rk、rk、r 是整数，且 0⩽r<p0 \leqslant r<p0⩽r<p，称 kkk 为 nnn 除以 ppp 的商， rrr 为 nnn 除以 ppp 的余数

      取模运算： a%pa \% pa%p （或 amodpa mod pamodp），表示 aaa 除以 ppp 的余数

    2. 性质

      1. (a+b)modp=(amodp+bmodp)modp(a + b) mod p = (a mod p + b mod p) mod p(a+b)modp=(amodp+bmodp)modp

      2. (a−b)modp=(amodp−bmodp)modp(a - b) mod p = (a mod p - b mod p) mod p(a−b)modp=(amodp−bmodp)modp

      3. (a∗b)modp=(amodp∗bmodp)modp(a * b) mod p = (a mod p * b mod p) mod p(a∗b)modp=(amodp∗bmodp)modp

        • (ab)modp=((amodp)b)modp\color{#f40} (a^b) mod p = ((a mod p)^b) mod p(ab)modp=((amodp)b)modp

      其中第三条性质很重要，对于后面 DH 算法的证明很关键

    # DH 算法

    # 步骤

    1. 第一步：选取两个全局公开的参数，一个素数 qqq 和一个整数 aaa，其中 aaa 是 qqq 的一个原根

    2. 第二步：用户 A 生成私钥和公钥，私钥保密，公钥公开

      1. 私钥：SAS_{A}SA​，它是一个随机数，并且满足 SA<qS_{A}<qSA​<q

      2. 公钥：PA=aSAmodqP_{A}=a^{S_{A}} mod qPA​=aSA​modq，其中 SAS_{A}SA​ 称为 PAP_{A}PA​ 的以 aaa 为基数的模 qqq 的离散对数

    3. 第三步：用户 B 生成私钥和公钥，私钥保密，公钥公开

      1. 私钥：SBS_{B}SB​，它是一个随机数，并且满足 SB<qS_{B}<qSB​<q

      2. 公钥 ：PB=aSBmodqP_{B}=a^{S_{B}} mod qPB​=aSB​modq，其中 SBS_{B}SB​ 称为 PBP_{B}PB​ 的以 aaa 为基数的模 qqq 的离散对数

    4. 第四步：用户 A 生成共享秘密密钥

      KA=(PBSA)modqK_{A}=(P_{B}^{S_{A}}) mod qKA​=(PBSA​​)modq

    5. 第五步：用户 B 生成共享秘密密钥

      KB=(PASB)modqK_{B}=(P_{A}^{S_{B}}) mod qKB​=(PASB​​)modq

    至此，用户 A 和用户 B 都生成了一个共享秘密密钥 KAK_{A}KA​ 和 KBK_{B}KB​，并且它们是相等的，WHY?

    # 证明

    如何证明共享秘密密钥 KAK_{A}KA​ 与 KBK_{B}KB​ 相等呢？

    已知

    KA=(PBSA)modqK_{A}=(P_{B}^{S_{A}}) mod qKA​=(PBSA​​)modq

    因为

    PB=aSBmodqP_{B}=a^{S_{B}} mod qPB​=aSB​modq

    所以

    KA=((aSBmodq)SA)modqK_{A}=((a^{S_{B}} mod q)^{S_{A}}) mod qKA​=((aSB​modq)SA​)modq

    又因为模运算的第三条性质

    (ab)modp=((amodp)b)modp\color{#f40} (a^b) mod p = ((a mod p)^b) mod p(ab)modp=((amodp)b)modp

    所以

    KA=(aSB)SAmodq=(aSA)SBmodq=(aSAmodq)SBmodqK_{A}=  (a^{S_{B}})^{S_{A}} mod q =  (a^{S_{A}})^{S_{B}} mod q = (a^{S_{A}} mod q)^{S_{B}} mod qKA​=(aSB​)SA​modq=(aSA​)SB​modq=(aSA​modq)SB​modq

    又因为

    PA=aSAmodqP_{A}=a^{S_{A}} mod qPA​=aSA​modq

    所以

    KA=(PA)SBmodq=(PASB)modqK_{A} = (P_{A})^{S_{B}} mod q = (P_{A}^{S_{B}}) mod qKA​=(PA​)SB​modq=(PASB​​)modq

    为什么很难破解共享秘密密钥 KAK_{A}KA​？

    攻击者可以获得的数据有 q、a、PA、PBq、a、P_{A}、P_{B}q、a、PA​、PB​

    要想由这些数据获取共享秘密密钥 KA=(PBSA)modqK_{A}=(P_{B}^{S_{A}}) mod qKA​=(PBSA​​)modq，就必须再知道 SAS_{A}SA​

    又因为 SAS_{A}SA​ 是 PAP_{A}PA​ 的以 aaa 为基数的模 qqq 的离散对数

    当 qqq 值很大时，离散对数 SAS_{A}SA​ 几乎不可求，这就是离散对数难解之题

    所以攻击者很难破解共享秘密密钥 KAK_{A}KA​

    同理，KBK_{B}KB​ 也很难破解

    # ElGamal 算法

    ElGamal 加密算法是基于 DH 密钥交换算法的非对称加密算法，其原理如下

    1. 密钥生成过程

      选择一个素数 ppp 以及两个小于 ppp 的随机数 ggg 和 xxx，计算

      y=gxmodpy = g^{x} mod py=gxmodp

      其中公钥 (y,g,p)(y,g,p)(y,g,p) ，私钥 xxx

      由离散对数的定义可知，xxx也称为 y、g、py、g、py、g、p 的离散对数

      因此，当素数 ppp 很大时，由公钥 (y,g,p)(y,g,p)(y,g,p) 推导出私钥 xxx 是极其困难的事

    2. 加密过程，使用公钥 (y,g,p)(y,g,p)(y,g,p)

      对于明文信息 MMM，随机选择一个与 p−1p-1p−1 互素的整数 kkk，计算

      C1=gkmodpC_{1} = g^{k} mod pC1​=gkmodp
      C2=ykMmodpC_{2} = y^{k} M mod pC2​=ykMmodp

      密文为 C=(C1,C2)C=(C_{1},C_{2})C=(C1​,C2​)

    3. 解密过程，使用私钥 xxx

      M=C2C1xmodpM = \frac{C_{2}}{C_{1}^{x}} mod pM=C1x​C2​​modp

    因为

    C2C1xmodp=ykMmodpgkxmodpmodp=ykMgkxmodp=ykMykmodp=Mmodp\frac{C_{2}}{C_{1}^{x}} mod p = \frac{y^{k} M mod p}{g^{kx} mod p} mod p =  \frac{y^{k} M }{g^{kx}} mod p =  \frac{y^{k} M }{y^{k}} mod p =  M mod pC1x​C2​​modp=gkxmodpykMmodp​modp=gkxykM​modp=ykykM​modp=Mmodp
